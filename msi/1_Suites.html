Suites numériques.

Suites numériques

  1. Introduction.

    Traditionnellement, une suite numérique est une suite infinie de nombres rangés ( l'ordre a une importance. ) Exemple : 1, 2, 3, 5, 8, 1, 0, 1.. // Suite des décimales des nombres irrationnels (\(\pi\), \(e\) ) (( 3.1415926545 => 3, 1, 4, 1, 5, 9..) On peut formaliser certaines suites, i.e. de les mettre sous une forme plus "condensée". ( Pas systématique, par ex pour une suite des naturels => \(u_n=n\)
    La suite se note \((u_n)\) ou \((u_n)_{n \in \mathbb N}\) La suite est ainsi totalement définie par son terme général : pour l'obtenir en tant qu'énumération on n'a qu'à itérer n. On peut remarquer une régularité dans les termes de la suite ( alternance de 0 et de 1 par exemple ) a partir d'un certain rang. Pour avoir \(u_n\) tu penses à une formule qui satisfasse cette alternance.
    Il se trouve que \(u_n\) = \(\dfrac{1}{2}(1-(-1)^n), \forall n \in \{1, 2, 3, 4, 5, 6\} \)
    Observation fondamentale : Les suites numériques sont des cas particulier de fonctions \(f:\mathbb R \mapsto \mathbb R\) au sous-ensemble \(\mathbb N\) (ou \(\mathbb Z\)) de \(\mathbb R\). Sur les exemples données on peut trouver des fonctions \(f:\mathbb R \mapsto \mathbb R \) telles que \(f_{\left|\mathbb N\right.} = u \) où \( u:\mathbb N \mapsto Im(f) \subset \mathbb R\) vérifie \(u(n) = u_n \)
    ( \(f_{\left|\mathbb N\right.} \) c'est la restriction d'une fonction sur un sous ensemble. // \(Im(f)\) -> Image de \(f\))
    La suite des entiers naturels peut être vue comme une correspondance : $$\mathbb N^* = \{1,2,3...\}$$
    et
    $$(u_n)_{n \in \mathbb N^*} \equiv \{1, 2, 3,..\}$$
    Donc la suite \( (u_n)_{n \in \mathbb N } \) n'est autre qu'une fonction \( \mathbb N^* \mapsto \mathbb R \) qui vérifie \( u(n) = u_n \). Dans ce cas \( u(n) = n \), donc une des fonctions \(f:\mathbb R \mapsto \mathbb R \) tel que \(f_{\left|\mathbb N^*\right.} = u \)est \(f(x) = x\).
    ( C'est marrant alors je le note : \( f(x)=0 \forall x \in \bigcup_{n=6}^{\infty} ]n, n+1[ \))

  2. Définir une suite. Différentes façons.

    • Cas 1 ( heureux )

      \(u_n\) a une formule explicite. Par exemple : La suite \( (u_n) : u_n = \frac{n^{2}+2n}{3} \)

    • Cas 2 (implicite heureux )

      On n'a pas de formule explicite pour (\(u_n\)), mais on a une forme implicite. Après un peu de boulot, on peut déduire une formule explicite pour (\(u_n\)). Par exemple $$ u_n = \sum_{k=0}^{n}k $$
      Démo à la Gauss, ça donne \( \frac{n(n+1)}{2} \)
      Dans des cas trop cool, on peut deviner une formule pour (\(u_n\)) et démontrer que c'est la bonne par récurrence.
